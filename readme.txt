The file main.py is the starting point of execution.
The decision tree implementation is packaged into the package “decisiontree”
Please unzip the zip file submitted.
In terminal, change the directory to the unzipped folder, and execute the command as shown below:

python main.py <L> <K> <training-set> <validation-set> <test-set> <to-print>

For example, for K = 20, L = 10, and to print the decision tree, execute the following command:  
python main.py 20 10 data_sets2/training_set.csv data_sets2/validation_set.csv data_sets2/test_set.csv yes
